# AI Translator

Obsidian 插件，基于大语言模型实现右键翻译选中文本。

## 功能

- 右键选中文本即可翻译
- 支持多种 API 提供商
- 可配置翻译目标语言

## 安装

1. 克隆本仓库
2. 执行 `npm install`
3. 执行 `npm run build`
4. 将 `packages/plugin/` 目录下的生成的main.js和manifest.json复制到 .obsidian/plugins/"ai translator" 插件目录

## 开发

```bash
npm run dev   # 开发模式
npm run build # 构建
```
